

# 🛒 Groceries E-Commerce App

A premium, fully animated grocery shopping application built with Flutter. This project demonstrates a production-ready mobile application featuring a modern UI/UX and a clean, modular codebase.

## ✨ Key Features
* **Premium UI/UX:** Clean, modern design featuring a custom curved navigation bar and crisp SVG icons for a pixel-perfect look.
* **Fluid Animations:** Beautiful staggered entry animations and page transitions using `flutter_animate` to bring the application to life.
* **Modular Components:** High reusability of UI elements like product cards, category cards, and custom inputs.

## 🛠 Tech Stack
* **Framework:** Flutter & Dart
* **Architecture:** Modular Layer-Based Structure
* **Key Packages:** `flutter_animate`, `curved_navigation_bar`, `flutter_svg`

## 📁 Folder Structure Highlight
The project is organized cleanly to separate UI screens, reusable components, and core configurations:
```text
lib/
 ├── core/          # App theme, colors, and styling configurations
 ├── screens/       # Full-page UI views (Home, Explore, Cart, etc.)
 ├── widgets/       # Reusable UI components (Buttons, Cards, TextFields)
 └── main.dart      # Application entry point
```

## 🚀 How to Run
1. Clone the repository:
   ```bash
   git clone <your-repo-link>
   ```
2. Navigate to the project:
   ```bash
   cd groceries_app
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run the app:
   ```bash
   flutter run
   ```
